## Project Overview

**Sherlock: Hidden Match-3 Cases** is a premium mobile mystery game that combines match-3 puzzle mechanics with hidden object gameplay. Over **2 years of development** (September 2022 - August 2024), I served as Level Designer and Technical Designer, contributing to creating engaging gameplay experiences for over **10 million players worldwide**.

## My Role & Responsibilities

### Level Design (Match-3)
- **Puzzle Design**: Created challenging and balanced match-3 levels with progressive difficulty curves
- **Mechanics Integration**: Designed levels that seamlessly incorporated special boosters and power-ups
- **Player Progression**: Developed level sequences that maintained player engagement while teaching new mechanics
- **Balancing**: Fine-tuned level difficulty through playtesting and data analysis

### Level Design (Hidden Object)
- **Scene Composition**: Designed intricate hidden object scenes based on classic literature themes
- **Object Placement**: Created natural yet challenging object hiding spots to maintain gameplay flow
- **Narrative Integration**: Ensured hidden object scenes supported the Sherlock Holmes storyline

### Technical Design
- **Systems Design**: Developed technical specifications for gameplay mechanics and features
- **Tool Development**: Created and maintained level design tools and workflows
- **Data Management**: Implemented systems for level progression, difficulty scaling, and player analytics
- **Cross-functional Collaboration**: Worked closely with programmers to implement complex gameplay features

## Project Achievements

- **Commercial Success**: 4.7/5 rating with 24,000+ reviews
- **Global Reach**: 10 million players worldwide across multiple platforms
- **Technical Excellence**: Smooth performance across iOS, Android, PC, and Mac platforms
- **Player Retention**: Designed progression systems that kept players engaged long-term

## Technical Challenges

### Match-3 Level Generation
- Developed algorithms for procedural level validation
- Created tools for efficient batch level testing and balancing
- Implemented systems for dynamic difficulty adjustment

### Cross-Platform Optimization
- Ensured consistent gameplay experience across different screen sizes and input methods
- Optimized level data for memory efficiency on mobile devices

### Analytics Integration
- Designed telemetry systems to track player behavior and level completion rates
- Created dashboards for real-time level performance monitoring

## Key Features Delivered

- **100+ Match-3 Levels**: Progressive difficulty with unique mechanics
- **Diverse Hidden Object Scenes**: Literary-themed environments with varying complexity
- **Integrated Progression**: Seamless connection between different gameplay modes
- **Adaptive Difficulty**: Dynamic systems that adjust to player skill level
